Python Calculator with Graphical Interface

A simple calculator application developed in Python using the Tkinter library for the graphical user interface.

Features

- Basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Clean and user-friendly graphical interface
- Clear button (C) to reset the display
- Error handling (e.g., division by zero)

Interface

The application includes:

- A display screen for input and results
- Buttons arranged in a standard calculator layout
- A dark theme for improved visual comfort


How to Run the Project

1. Prerequisites

- Python 3.x installed on your system

2. Clone the repository

git clone https://github.com/your-username/your-repository.git

3. Navigate to the project folder

cd your-repository

4. Run the application

python calculadora.py
